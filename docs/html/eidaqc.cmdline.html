eidaqc.cmdline module

Command line interface to eidaqc package

Syntax:

eida <method> <args> <configfile>

method:

  • avail: run availability test

  • inv: run inventory test

  • rep: create html and pdf report

  • templ: create templates of config file

and html-style file

param - avail

none

param - inv

request level for inventory. Any of ‘network’, ‘station’ or ‘channel’

param - rep

none

param - templ

none

configfile:
  • methods ‘avail’, ‘inv’, ‘rep’:

    mandatory, path to config file

  • method ‘templ’:

    optional; file name for default file. If not given file name will be “default_config.ini” in current dir.
